CSS Examples

A set of examples of new-ish CSS features.

Several assumptions are being made with these examples. One is that Sass is being used; the examples are CSS, but do occasionally reference variables. Second is that Autoprefixer is being used. The latter is extremely simple, Just Works, and is essential unless you really enjoy adding vendor prefixes by hand. There isn't a good reason not to be using it.
